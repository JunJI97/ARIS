## Project Overview

ARIS(Asset Risk Integrated System)는 금융자산(채권/주식)과 프로젝트 사업성을 한 화면에서 평가하는 과제 제출용 통합 가치평가 시스템이다. MVP의 핵심은 복잡한 예측 모델보다 금융 수학 기반 계산 로직의 정확성을 보여주는 것이다.

현재 코드는 Next.js frontend와 FastAPI backend의 최소 실행 구조가 준비된 상태이며, 앞으로 채권 PV/Duration/Convexity, 신용 리스크 스코어, 프로젝트 NPV/IRR/Payback, Parametric VaR 계산 엔진을 붙이는 방향으로 확장한다.

## Project Structure

이 프로젝트는 두 개의 주요 파트로 구성된다.

### Frontend (`frontend`)

- **Type:** Next.js App Router application

- **Tech Stack:** Next.js 16, React 19, TypeScript, Tailwind CSS 4, Recharts, Zustand
- **Entry Point:** `frontend/app/page.tsx`

### Backend (`backend`)

- **Type:** FastAPI API service
- **Location:** `backend/`
- **Tech Stack:** Python, FastAPI, Uvicorn
- **Entry Point:** `backend/main.py`

## Cross-Part Integration

Frontend는 사용자 입력 중심 대시보드를 제공하고, backend는 금융 계산 API를 제공한다. MVP에서는 다음 API 영역으로 확장하는 것이 자연스럽다.

- `/api/bonds/*`: 채권 현재가치, 듀레이션, 컨벡시티, 금리 시나리오
- `/api/credit-risk/*`: 재무비율 기반 신용점수와 리스크 등급
- `/api/projects/*`: NPV, IRR, Payback Period
- `/api/market-risk/*`: 포트폴리오 Parametric VaR
